# Docs Folder

This folder contains the final documentation and deliverables for the **Patient Churn Prediction and Healthcare Retention Analytics** capstone project.

## Contents

- `report.md` — Final project report including background, data analysis, model training, deployment, and conclusions.
- Presentation materials and supporting documentation (if added later).

## Project Overview

The project focuses on predicting patient churn using machine learning models such as:
- Logistic Regression
- Random Forest
- XGBoost

It also includes:
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Streamlit Web Application
- Healthcare retention analytics

## Technologies Used

- Python
- Pandas
- Scikit-learn
- XGBoost
- Plotly
- Streamlit
